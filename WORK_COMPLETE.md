## 🎯 How to Use

### GitHub to App Converter
```bash
cd /workspace
pip install -r requirements.txt
python run.py
# Open http://localhost:8000
```

### E-Commerce System
```bash
cd /workspace/generated_apps/ecommerce-product-browser
docker-compose up -d
npm install && cd frontend && npm install && cd ..
cp .env.example .env
npm run db:seed
npm run dev
# Frontend: http://localhost:3000
# Backend: http://localhost:3001
```

### Run All Tests
```bash
# GitHub Converter
pytest tests/ -v
python test_conversion.py

# E-Commerce
cd generated_apps/ecommerce-product-browser
npm run test:all
```
