VERIFICATION COMMANDS
================================================================================

To verify the installation:

1. Check test discovery:
   pytest --collect-only

2. Count tests:
   pytest --collect-only | grep "<Function" | wc -l

3. Run smoke tests:
   pytest -m smoke -v

4. Check coverage:
   pytest --cov=src --cov-report=term-missing

5. Use Make commands:
   make help
   make test
   make coverage-html
