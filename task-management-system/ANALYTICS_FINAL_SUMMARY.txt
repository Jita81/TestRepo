## QUICK START

1. Install dependencies:
   npm install puppeteer csv-writer bull node-cron chart.js react-chartjs-2

2. Run database migration:
   psql -U postgres -d task_management < database/analytics-schema.sql

3. Configure environment (.env):
   REDIS_HOST=localhost
   SENDGRID_API_KEY=your-key
   APP_URL=https://your-domain.com

4. Start server (scheduler auto-initializes):
   npm start

5. Access dashboard:
   Navigate to /projects/{projectId}/analytics
